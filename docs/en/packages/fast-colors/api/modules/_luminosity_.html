luminance

▸ luminance(targetLuminance: number, sourceColor: Chroma, round?: function): number[]

Defined in luminosity.ts:16

Adjust a color to a specific luminosity. This function is almost a direct copy of https://github.com/gka/chroma.js/blob/master/src/io/luminance.coffee, except that it accepts a rounding function. This is necessary to prevent contrast ratios being slightly below their target due to rounding RGB channel values the wrong direction.
